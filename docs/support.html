IFC Model Compare Support

Compare IFC model revisions with confidence.

IFC Model Compare helps construction, design and BIM professionals compare two IFC model revisions, review added, removed, changed and unchanged objects, inspect model data and export structured comparison reports.

Supported files

IFC Model Compare currently supports Industry Foundation Classes files using the .ifc file extension.

IFCZIP and IFCXML files are not currently supported.

How to compare two models

  1. Open IFC Model Compare.
  2. Import the original IFC as Model A — Baseline IFC.
  3. Import the revised IFC as Model B — Revised IFC.
  4. Wait for the comparison to complete.
  5. Review the results in the Compare, 3D and Data sections.
  6. Export the results as CSV, JSON, Markdown or PDF.

3D viewer

The app uses its modern That Open viewer by default. The Classic viewer remains available as a fallback from Settings or when a model cannot be prepared by the primary viewer.

Large IFC files can take longer to prepare on older devices. Keep the app open while conversion is in progress.

Troubleshooting

  • Confirm the selected file has the .ifc extension.
  • Allow the file to finish copying before leaving the app.
  • For large files, allow additional time for the first 3D conversion.
  • Use Retry if a temporary viewer error occurs.
  • Select Use Classic if the primary viewer cannot prepare a model.
  • Clear and re-import a model if the source file has been replaced.
  • Ensure sufficient free storage is available for model processing and cache files.

Privacy

IFC files are processed locally on your device and are not uploaded by IFC Model Compare. The embedded viewer uses runtime resources packaged inside the app.

Contact support

Include the following information when reporting a problem:

  • Device model
  • iOS, iPadOS or macOS version
  • IFC file size
  • Whether the issue affects Model A, Model B or both
  • A screenshot of any error message
